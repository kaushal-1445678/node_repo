🔹 1️⃣ What is Node.js? (Very Simple)


👉 Node.js allows JavaScript to run outside the browser.
Before Node:
JavaScript → only runs in browser
After Node:
JavaScript → runs on server
Example uses:
APIs
Backend servers
Real-time apps
Chat apps


🧠 Simple Example
In browser:
console.log("Hello");
In Node terminal:
node app.js
Same JavaScript runs on server.


🔹 2️⃣ Why Node.js is Popular
✔ Fast
✔ JavaScript everywhere (frontend + backend)
✔ Non-blocking architecture
✔ Large ecosystem (npm)

🔹 5️⃣ What is Express? (Very Important)
👉 Express is a framework for Node.js to build APIs easily.
Without Express:
Too much code
Complex routing
With Express:
✔ Easy server creation
✔ Easy APIs
✔ Routing system


🔹 6️⃣ Create Node Project
Create folder:
backend-app
Go inside folder:
cd backend-app
Initialize project:
npm init -y
This creates:
package.json


🔹 7️⃣ Install Express
npm install express
This installs Express inside:
node_modules


🔹 8️⃣ Create Express Server
Create file:
server.js
Code:
const express = require('express');

const app = express();

app.get('/', (req, res) => {
  res.send("Hello from server");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});


🔹 9️⃣ Run Server
node server.js
Terminal output:
Server running on port 3000
Open browser:
http://localhost:3000
Output:
Hello from server
🎉 Your backend server is working.


🔹 🔟 What is happening in this code?
Import express
const express = require('express');
Loads Express library.
Create app
const app = express();
Creates server instance.
Create route
app.get('/', (req, res) => {})
When user visits /, server responds.
Send response
res.send("Hello from server");
Sends message to browser.
Start server
app.listen(3000)
Server runs on port 3000.


🔹 11️⃣ Create Another Route
Example:

🔹 12️⃣ Real API Example
app.get('/users', (req, res) => {

  const users = [
    {id:1, name:"Rahul"},
    {id:2, name:"Amit"}
  ];

  res.json(users);

});
Visit:
localhost:3000/users
Output:
[
 { "id":1,"name":"Rahul"},
 { "id":2,"name":"Amit"}
]
This is a backend API.
