🔹 1️⃣ What is Node.js? (Very Simple)


👉 Node.js allows JavaScript to run outside the browser.
Before Node:
JavaScript → only runs in browser
After Node:
JavaScript → runs on server
Example uses:
APIs
Backend servers
Real-time apps
Chat apps


🧠 Simple Example
In browser:
console.log("Hello");
In Node terminal:
node app.js
Same JavaScript runs on server.

🔹 8️⃣ Create Express Server
Create file:
server.js
Code:
const express = require('express');

const app = express();

app.get('/', (req, res) => {
  res.send("Hello from server");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});


🔹 9️⃣ Run Server
node server.js
Terminal output:
Server running on port 3000
Open browser:
http://localhost:3000
Output:
Hello from server
🎉 Your backend server is working.


🔹 🔟 What is happening in this code?
Import express
const express = require('express');
Loads Express library.
Create app
const app = express();
Creates server instance.
Create route
app.get('/', (req, res) => {})
When user visits /, server responds.
Send response
res.send("Hello from server");
Sends message to browser.
Start server
app.listen(3000)
Server runs on port 3000.

🔹 1️⃣ What is an API? (Very Simple)


👉 API = Application Programming Interface
In simple words:
API allows frontend to talk to backend
Example flow:
Angular → API → Node server → Database
Example request:
GET /users
Server returns:
[
 { "id":1, "name":"Rahul" },
 { "id":2, "name":"Amit" }
]
🔹 2️⃣ What is REST API?


👉 REST = Representational State Transfer
It means APIs follow standard HTTP methods.
These methods are:
Method	Purpose
GET	Read data
POST	Create data
PUT	Update data
DELETE	Remove data

🔹 5️⃣ POST API (Create Data)


First add middleware:
app.use(express.json());
Now create POST API:
app.post('/users', (req, res) => {

  const newUser = req.body;

  users.push(newUser);

  res.json({
    message: "User added",
    users: users
  });

});
Example request body:
{
 "id":3,
 "name":"Kaushal"
}


🔹 6️⃣ PUT API (Update Data)


app.put('/users/:id', (req, res) => {

  const id = parseInt(req.params.id);
  const updatedName = req.body.name;

  users = users.map(user => {
    if(user.id === id){
      user.name = updatedName;
    }
    return user;
  });

  res.json(users);

});
Example URL:
PUT /users/1
Body:
{
 "name":"Rohan"
}


🔹 7️⃣ DELETE API


app.delete('/users/:id', (req, res) => {

  const id = parseInt(req.params.id);

  users = users.filter(user => user.id !== id);

  res.json({
    message:"User deleted",
    users:users
  });

});
Example:
DELETE /users/2


🔹 8️⃣ What is req and res?


(req, res)

🔹 9️⃣ What is req.params?


Used to read URL parameters.
Example route:
/users/:id
If URL is:
/users/5
Then:
req.params.id
Output:
5


🔹 🔟 What is req.body?


Used to read data sent from client.
Example request:
{
 "name":"Kaushal"
}
Access it using:

🔹 12️⃣ Complete Example Server


const express = require('express');

const app = express();

app.use(express.json());

let users = [
 { id:1, name:"Rahul"},
 { id:2, name:"Amit"}
];

app.get('/users',(req,res)=>{
 res.json(users);
});

app.post('/users',(req,res)=>{
 users.push(req.body);
 res.json(users);
});

app.put('/users/:id',(req,res)=>{

 const id = parseInt(req.params.id);

 users = users.map(user=>{
  if(user.id === id){
   user.name = req.body.name;
  }
  return user;
 });

 res.json(users);

});

app.delete('/users/:id',(req,res)=>{

 const id = parseInt(req.params.id);

 users = users.filter(user=>user.id !== id);

 res.json(users);

});

app.listen(3000,()=>{
 console.log("Server running on 3000");
});
